## 🎯 Bamboo Portfolio Integration

**Status:** Planned (Post v2 Stability)  
**Priority:** High  
**Target Release:** v3.0

### Overview
Integration with Bamboo trading platform to track holdings, monitor performance, and align portfolio with advisory signals generated by the stock pipeline.

### Core Requirements

#### 1. Portfolio Tracking
- **Holdings Management**
  - Track current positions (stock, quantity, average cost)
  - Support multiple platforms (Bamboo, others)
  - Position lifecycle: open, active, closed
  
- **Transaction History**
  - Record all buy/sell trades
  - Track: date, stock, action, quantity, price, fees
  - Link transactions to advisory signals (which trades followed recommendations)

- **Performance Monitoring**
  - Daily P&L tracking (unrealized gains/losses)
  - Position-level and portfolio-level metrics
  - Days held, cost basis, current value
  - Compare holdings vs current advisory signals

#### 2. Data Import Options
Need to determine which Bamboo supports:
- [ ] **API Integration** - Real-time sync if Bamboo has developer API
- [ ] **CSV/Excel Import** - Manual export from Bamboo, auto-import to pipeline
- [ ] **Email Parsing** - Parse trade confirmation emails
- [ ] **Manual Entry** - CLI commands for trade logging

#### 3. Features

**CLI Commands:**
```bash
# Log trades
portfolio add-trade NEIMETH buy 100 13.50 --fees 50 --notes "Entry after pullback"

# View holdings
portfolio view --format table

# Check performance
portfolio performance --export reports/portfolio.json

# Transaction history
portfolio transactions --stock NEIMETH --days 30

# Signal alignment
portfolio signals --holdings-only
```

**Database Tables:**
- `user_portfolio` - Current holdings
- `portfolio_transactions` - Buy/sell history
- `portfolio_performance` - Daily snapshots
- `trading_signal_log` - Track advisory signal execution

**Reports/Dashboards:**
- Portfolio summary with total P/L
- Holdings aligned/misaligned with current signals
- Trade performance vs advisory recommendations
- Signal hit rate tracking

#### 4. Use Case Workflow
1. Platform generates signal: "BUY NEIMETH - Score 85/100, Entry ₦13.50"
2. User reviews and decides
3. User executes trade manually in Bamboo app
4. User logs trade: `portfolio add-trade NEIMETH buy 100 13.50`
5. Platform tracks position and calculates P/L daily
6. Platform compares holdings against new signals
7. User gets alerts: "Your NEIMETH position up 15% - signal changed to HOLD"
